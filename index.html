Use a rich, fluent interface to build event driven sagas that are easy to deploy in a distributed system. Check out Automatonymous on GitHub for details.

Easy integration with multiple messaging frameworks including MSMQ and RabbitMQ. There is also limited community support for Azure Service Bus (official support coming later in 2014).
